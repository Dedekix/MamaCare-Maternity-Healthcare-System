# MamaCare Frontend

This is the React + Vite frontend for the MamaCare Maternal Health Care Management System.

For the full project explanation, setup steps, backend API overview, user roles, and development commands, read the main README at:

```text
../README.md
```

## Quick Start

```bash
npm install
npm run dev
```

The frontend calls the backend at `http://localhost:8080` by default. To change it, create a `.env` file:

```env
VITE_API_BASE_URL=http://localhost:8080
```
